What's the difference?
HTML
Hyper Text Markup Language		 Create the structure
  • Controls the layout of the content
  • Provides the structure of the web page design
  • The fundamental building bolck of any web page
CSS
Cascading Style Sheet		 Stylize the website
  • Applies style to the web page elements
  • Target various screen sizes to make web pages responsive
  • Primarily handles the "look and feel" of a web page
JavaScript Increase interactivity
  • Adds interactivity to a web page
  • Handles complex functions and features
  • programmatic code which enhances functionality
